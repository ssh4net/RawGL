API Reference
=============

This page is generated from the public C++ headers via Doxygen XML and Breathe.

The stable public surface is centered on:

- ``rawgl/rawgl.h``
- ``rawgl/rawgl_io.h``
- ``rawgl/rawgl_batch.h``
- ``rawgl/rawgl_cli.h``

``rawgl/rawgl_core.h`` is still present while the transitional graph-oriented
API remains public, but new code should prefer the higher-level façade headers.
