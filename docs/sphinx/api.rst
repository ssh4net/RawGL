API Reference
=============

This page is generated from the public C++ headers via Doxygen XML and Breathe.

The stable public surface is centered on:

- ``rawgl/rawgl.h``
- ``rawgl/rawgl_io.h``
- ``rawgl/rawgl_batch.h``
- ``rawgl/rawgl_cli.h``

``rawgl/rawgl_core.h`` remains available as the low-level graph-oriented API.
Use it when you need explicit graph control. Use the higher-level façade
headers for normal application code.
